AIM MIL-STD-1553

The AIM MIL-STD-1553 Custom Device configures AIM hardware designed for avionics communications using the MIL-STD-1553 protocol. This addon configures only one BIU (Bus Interface Unit) of an AIM device. You may use multiple AIM MIL-STD-1553 addons in your VeriStand project.

Device Configuration

This device has the following configuration settings:

  • Name—Specifies the custom device name.
  • Description—Documents this instance of the custom device.
  • Initialization Method—Specifies whether to reference the hardware device by PXI Slot Number or Card Number.
  • PXI Slot Number—Specifies the PXI Slot Number of the hardware device.
  • Card Number—Specifies the card identifier as enumerated by the AIM hardware API.
  • BIU Number—Specifies the device's BIU (Bus Interface Unit) to use for communication.
  • Coupling Mode—Specifies the physical MILbus coupling mode.
  • Bus Selection—Specifies the bus for which to configure the coupling mode.
